# bsp - shadowsocks多用户、限流管理工具

**bsp是shadowsocks多用户、限流管理工具** 

支持shadowsocks-python和shadowsocks-Go

 **安装** 

```
git clone https://github.com/edboffical/bsp.git
cd bsp
chmod 775 install
./install
```

详细使用请查阅使用文档 [wiki](https://github.com/edboffical/bsp/wiki)
